API instructions:

    1. Head to this website: https://developer.edamam.com/edamam-recipe-api
    2. Use the free version API to obtain your unique key and ID
    3. Replace your unique key and ID in the index.js file

Server Run Instrctions:

    1. Open terminal in folder where server.js is located
    2. Type the following command in the terminal: node server.js
    3. Open a browser with the following url: http://localhost:3000/index.html

Functionality:

    Add your desired ingredients, by typing in the text box, and press enter or the "Add Ingredient" button
    Use the slider to set your desired amount of recipes to show
    See you added ingredients in the list section
    Choose any further recipe restrictions on the bottom section under "Advanced Selection"
    Press "Find Recipes" and the list of the recipes found will appear, and you can click on the recipe name to be brought to the recipe page
